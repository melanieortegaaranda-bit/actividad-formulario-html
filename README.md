# actividad-formulario-html
Descripción

Este proyecto consiste en el desarrollo de un formulario web de inscripción para un Taller de Desarrollo Web utilizando únicamente HTML. El formulario permite recopilar información personal, académica y preferencias de los estudiantes interesados en participar en el taller.

Objetivo

Aplicar los conocimientos sobre formularios HTML mediante el uso de diferentes controles de entrada y atributos de validación para crear una interfaz de registro funcional.

Características del formulario

Nombres y apellidos
Correo electrónico
Teléfono
Edad
Fecha de nacimiento
Datos académicos:
Carrera o curso
Nivel académico
Institución educativa
Preferencias del taller:

Temas de interés

Disponibilidad:
Fecha de inicio

Jornada preferida
Información adicional:
